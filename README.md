# 🐱 Catris
**A Purr-fectly Stacked Tetris Clone**

Catris is a lightweight, cat-themed Tetris clone built with vanilla JavaScript and HTML5 Canvas. It was designed as a portfolio piece for Sophi Thrice Saitta to showcase clean, efficient game logic and creative UI integration.

## 🎮 Features
- **Classic Tetris Mechanics:** Standard 10x20 grid, all 7 tetromino shapes, line clearing, and increasing difficulty.
- **Cat Theme:** Blocks are represented by unique cat breeds/emojis.
- **Next Piece Preview:** See which cat is coming next to plan your strategy.
- **Responsive Controls:** Optimized for desktop play with hard-drop support.
- **Speed Scaling:** The game gets faster every 5 lines cleared!

## ⌨️ Controls
- **Left Arrow:** Move Left
- **Right Arrow:** Move Right
- **Up Arrow:** Rotate Piece
- **Down Arrow:** Soft Drop
- **Spacebar:** Hard Drop (Instant)

## 🛠️ Tech Stack
- **HTML5 Canvas:** For high-performance 2D rendering.
- **Vanilla JavaScript:** No frameworks, just pure logic.
- **CSS3:** Cyberpunk-inspired styling with a feline twist.

## 🚀 Setup & Deployment
1. Clone the repository:
   ```bash
   git clone https://github.com/STS-369/Catris.git
   ```
2. Open `index.html` in any modern web browser.
3. **GitHub Pages:** This project is 100% compatible with GitHub Pages. Simply enable it in your repository settings to host it live!
